Read full abstract

We introduce Vid2Sid, a vision-language model (VLM) guided system identification pipeline that calibrates simulator physics parameters by comparing paired simulation and real-robot videos. Accurate system identification is a prerequisite for sim-to-real policy transfer, yet existing methods rely on black-box optimization that provides no insight into which physical discrepancies drive error. Vid2Sid decomposes the problem into two layers: a perception layer that extracts trajectories from video using foundation models (SAM3 segmentation or marker tracking), and a reasoning layer where a VLM (Gemini 2.5 Pro) diagnoses dynamic mismatches and proposes targeted parameter corrections with natural-language rationales. We evaluate on two morphologically distinct platforms (a rigid tendon-actuated finger in MuJoCo with 6 parameters, and a soft continuum tentacle in PyElastica with 8 parameters) in both sim-to-sim and sim-to-real settings with holdout control profiles. Vid2Sid achieves the best average rank (1.7) across all sim-to-real settings, recovers physically correct parameters in sim-to-sim (under 13% relative error vs. 28-98% for baselines), and provides interpretable physics diagnostics at every iteration.
